### Keyboard Shortcuts
| Shortcut        | Command              |
| --------------- |:-------------:       |
| Toggle Sidebar      | ctrl ;           |
| Change View      | ctrl alt (1-4)      |
| Change Media Source | ctrl (1-9)       |
| Delete Item | backspace                |
| Expand Item | spacebar                 |
| Choose Item | return                   |
| Rename Item | ctrl r                   |
| Filter Items | ctrl f                  |
